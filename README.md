# ACLI-Project

Checks if you're logged into Rovodev ACLI and launches it automatically.

## How to use

```bash
python run.py
```

## What it does

- If you're logged in: Shows your info and opens ACLI
- If you're not logged in: Opens ACLI and prompts you to log in

## Requirements

- Python 3.6+
- PyYAML (`pip install pyyaml`)
- ACLI installed
